В конце 2017 года Mozilla планирует убрать из FireFox возможности, которые использует Http UserAgent Cleaner.
Новые возможности (в WebExtensions) недостаточны. Возможно, разработчик прекратит разработку данного дополнения.
(пользуйтесь tor browser https://www.torproject.org/ )

По умолчанию дополнение почти ничего не делает, требует терпеливой настройки. Пожалуйста, смотрите справку по адресу http://huac.8vs.ru/ (закройте рекламу бесплатного хостинга, если она появилась) .

Требует отключения <a href=http://wiki.mozilla.org/Electrolysis>E10S</a>

* Оценивает безопасность https (TLS) соединения, включая безопасность всех побочных запросов со страницы
* Позволяет следить за сменой сертификатов на сайтах
* Позволяет блокировать подгрузку не https-страниц
* Следит за изменением сертификатов страниц https-сайтов
* Повышает приватность путём подмены "отпечатка браузера". Может случайно посылать комбинации UserAgent, locale, timezone, canvas. Отключает (с настройкой под домену) WebRTC, ajax, куки.
* Блокирует "resource://" запросы для повышения приватности
* Блокирует запросы к сторонним сайтам и позволяет управлять куками, отсылаемыми к сторонним сайтам
* Позволяет просмотреть все корневые сертификаты в едином списке и оценить число доверенных корневых сертификатов
* Позволяет управлять включёнными криптографическими примитивами
* Повышает безопасность включением небольшой антифишинговой функции - фильтра "Пароль" (см. справку)

Версия, распространяемая через галерею - не полная. Полная (оригинальная) версия по адресу http://huac.8vs.ru/ .

Преимущества версии для галлереи:
1. Проверена независимыми редакторами галереи дополнений Mozilla на отсутствие вредоносного кода
2. Большая стабильность, так как дополнение прошло независимое тестирование

Недостатки версии для галлереи,
1. Полностью убран фильтр TimeZone, подменяющий дату и время на другой часовой пояс; убран индикатор, что extensions.blocklist настроен по умолчанию
2. Возможны очень серьёзные задержки с обновлением дополнения (до года), которые могут повлечь за собой долговременный отказ функций дополнения на новых версиях FireFox
3. Не осуществляется поддержка

Преимущества оригинальной версии:
1. Обновление происходит с меньшими задержками
2. Доступна полная функциональность вне зависимости от причуд редакторов галереи дополнений Mozilla
3. Возможна меньшая стабильность дополнения, но дефекты исправляются быстрее.

At the end of 2017 year Mozilla plans to remove some opportunities from FireFox that use the Http UserAgent Cleaner.
New opportunities (in WebExtensions) is insufficient. Maybe the developer will stop developing the Extension.
(use tor browser https://www.torproject.org/ )

By default, the Extension does almost nothing. The Extension requires the neat and complex configuration by user.
For get help in the your language, you can use <a href="https://translate.google.ru/translate?hl=en&sl=ru&tl=en&u=http%3A%2F%2Fhuac.8vs.ru%2F&sandbox=1">Google translate</a>  (original help in Russian language) (close the ads of free hosting, if it appeared).
Original version of extension available at http://huac.8vs.ru/ .

Need to disable <a href=http://wiki.mozilla.org/Electrolysis>E10S</a>.

The Extension:

* Evaluates the security strong of https (TLS) connections, including the strong of the all side requests from the page
* Allows you to monitor the certificate changes on the websites
* Allows you to block the https pages loading
* Monitors changes in the certificate of the https pages of sites
* Increases privacy by substitution of the "browser fingerprint". Can accidentally send a combination UserAgent, locale, canvas, fonts. Disables (with the setting under domain), WebRTC, ajax, cookies. In original version have substitution of the TimeZone fields.
* Blocks "resource://" requests for privacy reasons
* Block requests to third-party websites and allows you to manage cookies sent to third-party websites
* Allows you to view the all root certificates in a single list and to estimate the number of trusted root certificates
* Allows you to control the included cryptographic primitives
* Increases security by a small antiphishing function (see "password" filter in help)
